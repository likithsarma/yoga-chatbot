# Yoga ChatBot Requirements

##Objective
Provide accurate answers to yoga related FAQs inside the mobile app and guide users to useful video categories.

## Imports
User question text from app frontend

## Outputs
Text response with best matched answer or fallback

## Features (Include in this version)
1. Answer about 30 yoga FAQs using stored knowledge
2. Suggest categories like Beginner, Weight Loss, Flexibility, Stress Relief
3. Fast response using vector search
4. Deployed as a backend API for integration by frontend team

## Not Included(Future work)
1. Personalised recommendations
2. Workout scheduling
3. User profile understanding
4. Real time video navigation

## Success Criteria
80 percent of questions should match correct FAQ
Response time less than 2 seconds
Can handle basic user queries without errors
